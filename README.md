# QuizNova Pro

QuizNova Pro is a premium React + Vite quiz application built for a web development internship project. It features a modern glassmorphism UI, animated experience, categorized quiz questions, a timer-based quiz flow, results with score summary, and a dashboard overview.

## Features

- Modern landing page with animated hero section
- Category-based quiz selection
- Interactive quiz experience with progress tracking and timer
- Result summary with score, accuracy, skipped count, and time taken
- Dashboard for performance insights
- Theme toggle and sound toggle with local storage persistence
- Responsive design for desktop and mobile

## Tech Stack

- React
- Vite
- React Router DOM
- Framer Motion
- React Icons
- CSS

## Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open the local URL shown in the terminal

## Build

To create a production build:

```bash
npm run build
```

## Project Purpose

This project demonstrates advanced React concepts such as routing, reusable components, state management, animations, and polished UI design for a professional portfolio-level application.
